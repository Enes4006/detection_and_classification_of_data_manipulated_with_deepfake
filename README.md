🎭 Deepfake Video Detection with Deep LearningThis project, developed as part of the Computer Engineering Design Course at Yozgat Bozok University, focuses on detecting deepfake videos using deep learning techniques. It serves as a foundational implementation (Phase 1) for a more advanced system that will incorporate state-of-the-art architectures.📌 Project OverviewWith the rapid advancement of AI-generated media, detecting manipulated videos has become a critical problem. This project aims to classify videos as REAL or FAKE using a CNN-based pipeline designed to be scalable and ready for advanced enhancements.🛠️ Current Implementation (Phase 1)The current system follows an end-to-end workflow:Video → Frame Extraction → Face Detection → Preprocessing → Data Augmentation → CNN Model → Prediction 🔹 Key Steps:🎥 Pipeline Setup: Automated video-to-frame extraction using OpenCV.😐 Face Detection: Integrated face detection and cropping to focus only on relevant features.🖼️ Preprocessing: Standardized resizing and normalization (0-1 range).🔁 Data Augmentation: Implemented via the Albumentations library to improve robustness: Horizontal Flip Rotation Zoom & Random Cropping 🧠 CNN Architecture: A multi-layer Convolutional Neural Network built for initial feature extraction.⚖️ Model Calibration: Investigated "Majority Class Bias" issues and implemented custom threshold optimizations to balance REAL/FAKE detection.⚙️ Technologies UsedPython TensorFlow / Keras & PyTorch OpenCV NumPy & Pandas Matplotlib & Seaborn Scikit-learn ⚠️ Challenges & Performance Analysis📊 Current StatusThe applied L2 Regularization and Dropout techniques successfully mitigated overfitting, as observed in the training logs. However, performance is limited by the current dataset size (10 videos), which is insufficient for the model to generalize deepfake patterns effectively.🚩 Key Challenges:Data Scarcity: Limited dataset makes deep feature generalization difficult.Overfitting Risk: High variance due to small sample size.Computational Constraints: Hardware limitations during large-scale frame extraction.Environmental Noise: Variability in lighting and irrelevant frames in source videos.Technical Insight: The project's success is expected to increase linearly with the expansion of the dataset (e.g., 100+ videos), providing a larger feature space for the model to learn from.✅ Optimization Techniques✔ Batch Normalization: Used for stable and faster training.✔ Dropout (0.5): Implemented to prevent reliance on specific neurons.✔ L2 Regularization: Added to penalize large weights and reduce overfitting.✔ Early Stopping: Monitors validation loss to halt training at the optimal point.🚀 Roadmap (Future Work)This baseline system is the first step toward a Multimodal Hybrid Architecture. Future phases include:🔥 Transfer Learning: Transitioning to EfficientNet (B0-B4) and ResNet backbones for superior feature extraction.📊 Dataset Expansion: Integrating large-scale datasets like FaceForensics++, DFDC, and Celeb-DF.🎥 Temporal Modeling: Incorporating LSTM / GRU or 3D-CNNs to detect temporal inconsistencies across video frames.📡 Frequency Analysis: Adding an FFT (Fast Fourier Transform) stream to detect artifacts in the frequency domain.📱 Optimization: Model compression for real-time performance on mobile or embedded platforms.📌 ConclusionThis project represents an initial, successful step toward building a robust deepfake detection system. While current results are constrained by data volume, the architecture is modular, scalable, and engineered to support advanced deep learning enhancements in the next phase.
